Akcja 2 Dodawanie nowych stron do nawigacji - sposób 2

Tworząc nowy obiekt NavigationPage, podajemy mu w konstruktorze oprócz właściwości także opcjonalny obiekt klasy Navigation. 
    $newPage = new NavigationPage(
        array(
                'id' => 'firma_siedziba',
                'parentId' => 'firma',
                'label' => 'Prezentacja siedziby',
                'url' => '#'
        ), $navigation);  // $navigation jest obiektem klasy Navigation, na którym pracujemy
